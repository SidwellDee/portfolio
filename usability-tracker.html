Overview

To enhanve the monitoring and evaluation of the Client Management Information System (CMIS) adoption and performace, I developed a real-time Usability Tracker that provides actionable insights into system utilization, uptime and connectivity status across health facilities. The tracker plays a critical role in ensuring the reliability, availability and effectiveness of CMIS allowing project staff to tack how well facilities are leveraging the system. This system delivers delivers performance data overtime through dashboards which enables proactive system maintenance.

Key Contributions

  • Real-Time System Monitoring & Uptime Tracking - Developed a real-time system uptime tracker to monitor system availability and performance across all connected facilities.
  • System Utilization Analytics - Developed usage tracking mechanisms to capture visit counts and other variables that enables the measurement of system engagement rate across facilities.
  • Historical Reporting - Developed automated reporting tools to generate historical uptime data and facility and regional level usage trends.

Tech Stack
